## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Configuration

1. Create a `.env` file in the project root:

```
# Application
PORT=port
JWT_SECRET=key
NODE_ENV=env
APP_URL=url

# MongoDB
MONGODB_USERNAME=mongo_username
MONGODB_PASSWORD=mongo_password
MONGODB_HOST=mongo_host
MONGODB_PORT=mongo_port
MONGODB_DATABASE=mongo_db

# AWS
AWS_ACCESS_KEY_ID=id
AWS_SECRET_ACCESS_KEY=key
AWS_REGION=region
AWS_S3_BUCKET_NAME=bucket

# Python
PYTHON_HOST=port

# Email Service
EMAIL_HOST=host
EMAIL_PORT=port
EMAIL_SECURE=false
EMAIL_USER=user_email
EMAIL_PASSWORD=password
EMAIL_FROM=email_from
EMAIL_FROM_NAME=name

```
